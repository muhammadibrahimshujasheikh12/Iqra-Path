## üöÄ Getting Started

### Prerequisites
- Node.js (v18+)
- A Google AI Studio API Key
- A Supabase Project (SQL schema provided in `supabase_schema.sql`)

### Installation
1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/IqraPath.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Configure Environment Variables:
    Create a `.env` file with your `API_KEY` (Gemini) and Supabase credentials.

4.  Run the application:
    ```bash
    npm run dev
    ```
